# pomo
A cross-platform pomodoro timer terminal app.

```
   \/_____/      \/___/              \/___/       \/___/

    [g]o   [p]ause   [r]estart   [m]ute   [s]ettings

    > _
    
```

## features
```
- timed focus sessions
- timed breaks
- specify durations for focus + breaks
- audio alerts
- specify session repetition
```
